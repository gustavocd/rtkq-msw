# RTK Query + msw ![](https://github.com/gustavocd/rtkq-msw/workflows/rtkq-msw/badge.svg)

App example about testing `http` calls using `RTK Query` and `msw`.

## Install dependencies

```bash
make deps
```

## Running the app

```bash
make dev
```

Go to http://localhost:5173/

## Running the tests

```bash
make test
```
